# Ventoy-Theme-Terminal-Raccoon
Theme for Ventoy (made during 1.1.11) Should work for older and newer versions.

## Edit - security note

The author says ventoy is open-source, but the GitHub package contains pre-compiled blobs that are not source code (can't be audited, may be copyright-breaking/ break the licence ventoy is apparently under...) it works really well, but maybe Rufus should be used for security...

## Install

Drop the folder "ventoy" into the root of your USB stick with Ventoy installed onto it.
